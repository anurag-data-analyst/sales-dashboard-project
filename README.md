## Power BI Sales Dashboard

### Overview
This project analyzes mobile sales data using Power BI to uncover key business insights and trends.

### Tools Used
Power BI | Excel

### KPIs
- Total Sales: 769M
- Total Quantity: 19K
- Transactions: 4K

### Insights
- Sales trends by brand and mobile model
- Customer rating analysis
- Monthly sales performance tracking
