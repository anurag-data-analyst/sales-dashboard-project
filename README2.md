# 📊 Hotel Booking Cancellation Dashboard (Excel)

This project presents an interactive Excel dashboard analyzing hotel booking and cancellation trends.

## 🔍 Overview
The dashboard provides insights into booking status, cancellation rates, and customer behavior using charts and KPIs.

## 📌 Features
- Cancellation vs Confirmed Bookings (Pie Charts)
- Monthly Booking Trends (Bar Chart)
- Customer Segmentation Analysis
- Interactive Dashboard Design

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts (Bar, Pie)
- Data Cleaning & Formatting

## 📈 Key Insights
- High cancellation rate observed in certain months
- Majority of bookings are from specific customer segments
- Trends help identify peak booking periods
